## Основы docker

### Задание

1. Откройте VirtualBox manager, убедитесь, что все виртуальные машины выключены
2. В терминале выполните команду
    
    > docker-machine start
3. Убедитесь, что запустилась виртуальная машина с docker. После работы не забудьте выключить docker на учебном компьютере командой
    
   >docker-machine stop
4. С помощью команды 
   
   > docker images

    посмотрите имеющиеся в docker образы.
5. С помощью команды 
   
   > docker load -i doodle_cheers.docker

   загрузите образ из файла `doodle_cheers.docker`. Убедитесь, что в docker появился образ `cheers2019`.
6. С помощью команды 
   
   > docker run -it cheers2019

   запустите этот образ и терминал перейдет на работу с терминалом этого контейнера, который запущен этим образом. Закройте приветствие клавишей `Esc`.
7. С помощью команды 
   
   > docker run --help
   
   разберитесь, что означают ключи команды `run`, используемые в предыдущем примере.
8. Запустите еще одни терминал, в нем наберите команду
   
   > docker-machine ssh

    Должен запуститься терминал виртуальной машины -- терминал системы Linux, на которой запущен docker. Выполните в этом терминале команды для просмотра образов docker и запустите образ cheers2019. При запуске образа с помощью ключа --name укажите имя контейнера. 
10. На закрывая приветствие на терминале виртуальной машине, в другом терминале (терминале windows) выведите список контейнеров 
    
    > docker ps

    Определите, какая информация о контейнерах выводится этой командой.
11. С помощью команды 
    
    > docker rm -f имя_контейнера
    
    остановите запущенный контейнер. Выведите список контейнеров, посмотрите, как завершение контейнера отразиться в окне терминала виртуальной машины
