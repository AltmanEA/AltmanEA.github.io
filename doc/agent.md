# Правила для ИИ-агента

## Команда пользователя
> "обнови папку doc по правилам из agent.md"

## Алгоритм

### Шаг 1: Сканирование
Просканируй папку `doc/` рекурсивно (`ls` с `recursive: true`).

### Шаг 2: Определение изменений
Для каждой подпапки:
- Сравни текущий `index.html` со списком файлов в папке
- Определи:
  - какие файлы нужно **добавить** (есть в папке, нет в index.html)
  - какие файлы нужно **удалить** (есть в index.html, нет в папке)

### Шаг 3: Обновление index.html подпапки
Для каждой подпапки, где есть изменения:
1. Перечитай текущий `index.html`
2. **Сохрани заголовок** — возьми текст из существующего `<h1>...</h1>` (пользователь мог изменить "standart" на "Стандарт")
3. Сгенерируй новый список `<li>` элементов:
   - добавь новые файлы
   - удали отсутствующие
4. Отсортируй ссылки **алфавитно**
5. Собери HTML с **сохранённым заголовком**
6. Запиши файл

### Шаг 4: Обновление корневого index.html
- Проверь, соответствуют ли ссылки на подпапки реальным папкам
- Добавь новую подпапку, если она появилась
- Удали ссылку на удалённую подпапку

## Шаблон index.html подпапки

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>название_папки</title>
</head>
<body>
    <h1>Стандарт</h1>  <!-- этот заголовок сохраняй при обновлении -->
    <ul>
        <li><a href="файл1.расширение">файл1.расширение</a></li>
    </ul>
</body>
</html>
```

## Кастомные названия папок

Если нужно добавить новую папку с названием, отличающимся от имени папки на диске:

| Папка на диске | Заголовок в index.html |
|----------------|------------------------|
| standart       | Стандарты              |
| vibe           | ИИ-программирование    |

При создании нового index.html используй этот маппинг. Заголовок `<h1>` бери из таблицы или спроси у пользователя.

## Шаблон корневого index.html

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>doc</title>
</head>
<body>
    <h1>doc</h1>
    <ul>
        <li><a href="папка1/index.html">папка1</a></li>
        <li><a href="папка2/index.html">папка2</a></li>
    </ul>
</body>
</html>
```

## Важно

- Имя в `href` должно **точно** совпадать с именем файла на диске
- Используй только `read_file` и `ls` для анализа — не предполагай содержимое
- Если подпапка пустая — всё равно создай index.html с пустым списком `<ul></ul>`
