Замечание

Для корректного проведения документ нужно скорректировать (по сравнению с вариантом из предыдущей части). Имя реквизита «Движение» совпадает с именем служебной переменной, которую создает конструктор движения. Этот реквизит следует переименовать (используйте имя из двух слов, которое более точно опишет назначение этого реквизита).

Задание к лабораторной работе

  1. Повторите примеры из лекций.
  2. Реализуйте проведение документа для отчета преподавателя о выполненной нагрузке.
