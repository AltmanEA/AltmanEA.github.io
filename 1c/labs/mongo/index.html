3. Основы MongoDb

Для выполнения этой и некоторых последующих лабораторных работ рекомендуется использовать настроенную виртуальную машину с MongoDb или контейнером docker. Для виртуальной машины не забудьте открыть порт для MongoDb командой sudo ufw allow 27017.

Проект с примерами из лекций находится на странице https://github.com/AltmanEA/edu-kmongo.
