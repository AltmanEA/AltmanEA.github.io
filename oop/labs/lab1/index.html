Лекция

Задание

  1. Установите и настройте на своем компьютере среду разработки.

  2. Напишите функцию для вычисления площади круга по его адресу.

    Подсказки:

    • аргумент и результат должны быть типа Double;
    • импортируйте константу PI из библиотеки kotlin.math.

  3. Напишите тесты для этой функции.

  4. Переименуйте функцию с помощью рефакторинга. Отформатируйте текст с помощью функции среды reformat.

  5. Прогоните тесты по шагам в режиме отладки.
