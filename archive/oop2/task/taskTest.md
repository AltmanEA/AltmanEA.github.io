## Рейтинг участников

### Данные для обработки

Все вопросы имеют поля id и name, id вопросов уникально в рамках всего приложения (включая id группы вопросов). При добавлении в тест группы вопросов пользователю задается случайный вопрос из группы. Описание вопросов [тут](https://docs.moodle.org/38/en/Moodle_XML_format).

- вопросы Multiple choice
- вопросы True/false
- вопросы Short answer
- вопросы Numerical response
- вопросы Matching
- группа вопросов
  - id
  - название
  - [id_вопросов]
- тест
  - id
  - название
  - [id_вопросов]
  - шкала оценивания

### Варианты заданий

Для данных, редактирование которых не предусмотрено в варианте задания, нужно предусмотреть достаточно большой набор тестовых данных. При необходимости работы со страницами из другого варианта эти страницы создаются в виде тестового HTML (без интерактивных элементов).

1. Редактирование вопросов
  - вопросы Multiple choice
  - вопросы True/false
  - экспорт и импорт вопросов в формат Moodle XML
2. Редактирование вопросов
  - вопросы Short answer
  - вопросы Numerical response
  - экспорт и импорт вопросов в формат Moodle XML
3. Редактирование вопросов
  - вопросы Matching
  - группа вопросов
  - экспорт и импорт вопросов в формат Moodle XML
4. Редактирование тестов
  - список тестов
  - редактор теста
  - редактор шкалы оценивания (в том числе нужно разработать структуру данных для оценки)
5. Прохождение тестов
  - страница с выбором теста (выбор теста из списка и количества вопросов на странице)
  - прохождение теста (постранично)
  - страница с результатами теста
