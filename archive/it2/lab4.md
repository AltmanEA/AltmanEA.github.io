## Основы git

### Задание

1. Выполните задания из учебника [githowto](https://githowto.com/). Задания рекомендуется выполнять в linux-терминале контейнера cmake с настроенным общим с windows каталоге. Редактировать файлы лучше с помощью visual studio code (через общий каталог).
2. Для выполнения следующей части работы потребуется запустить контейнер с gitbucket сервером и два контейнера cmake.  Разные контейнеры cmake отобразите на разные папки физического компьютера и создайте в них разных пользователей git. Для доступа к серверу из других контейнеров используйте ip адрес компьютера.
   1. Создайте на сервере репозиторий. 
   2. Создайте в каталоге первого пользователя репозиторий, добавьте файл README, выложите его в созданный на сервере репозиторий (можно использовать имя пользователя и пароль root).
   3. Под вторым пользователем измените файл README. Сделайте коммит и выложите его на сервер.
   4. Под первым пользователем командой `git pull` загрузите с сервера изменения. Убедитесь, что загрузился текст второго пользователя.
   5. Измените одинаковую строчку файла под первым и вторым пользователем. Сделайте коммиты. Загрузите данные на сервер сначала под одним пользователем, потом по другим. Разрешите конфликт, возникший у второго пользователя, выгрузите результат на сервер.
