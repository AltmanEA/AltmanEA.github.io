Задание

Повторите пример из лекций.

Переделайте пример с использованием варианта класса Student, сделанного в 1 лабораторной работе.

Добавьте в API сервера точку входа “/groups/”, которая бы позволяла с помощью метода GET получить список групп, и точку “/students/group/{group}”, которая бы позволяла с помощью метода GET получить список студентов этой группы.

Факультативное задание

Реализуйте в API возврат списка студентов группы, при этом группа задается в виде аргументов формы (?group=29з).

Реализуйте в API возврат списка студентов группы, при этом группа передается в теле запроса.
