5. Аутентификация и авторизация

Задание

Реализовать REST API и клиент для редактирования информации на сервере (информация определяется по вариантам).

Номер варианта определяется как остаток от деления количества букв в фамилии студента на 2.

0 вариант: список пользователей (userList).

1 вариант: список ролей (userRoles).

Факультативное задание

Реализовать REST API и клиент для редактирования списка ролей, имеющих доступ к точками API студента.
