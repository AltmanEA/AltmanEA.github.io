- Результаты работ

  - разрабатываются студентом

#### Использование формулы расчета рейтинга

При расчете рейтинга выполняется цепочка правил. На вход первого правила подается 0, на входе второго -- результат первого, итоговый результат это результат выполнения последнего правила

Результат выполнения правила зависит от его типа:
- взвешенная сумма: ко входу прибавляется произведение id_результата и числа;
- условие входа (6 штук): если выполняется условие для балла из id_результата относительно числа, то результат равен входу, иначе 0.
- условие результата (6 штук): если выполняется условие для входа относительно числа, то результат равен баллу из id_результата
- коэффициент: вход умножается на коэффициент.
  
#### Использование формулы расчета рейтинга

Все три массива одинакового размера. Условие -- одно из 6 условий сравнения. Все условие проверяются по порядку, сравнивая значение на входе и число из второго массива. Если условие выполняется, что дальнейшая проверка не выполняется и результатом является цвет с индексом равным индексу сработавшего условия.

### Варианты заданий

Для данных, редактирование которых не предусмотрено в варианте задания, нужно предусмотреть достаточно большой набор тестовых данных. При необходимости работы со страницами из другого варианта эти страницы создаются в виде тестового HTML (без интерактивных элементов).

1. Редактирование исходных данных приложения

  - страница с группами (добавить/удалить группы);
  - страница со студентами группы (добавить/удалить студента);
  - страница с видами работ группы (добавить/удалить виды работ).

2. Ввод результатов работ

  - страница с таблицей студенты-вид работ, ввод даты и оценки;
  - редактирование дат занятий (записываются в вид работ)
  - страница с таблицей студенты-даты занятий (виды работ по названию, например - лабы), ввод вида работы и оценки;

3. Редактирование правил расчета рейтинга

  - страница с правилами расчета;
  - страница с формулами расчета;

4. Редактирование цветовой схемы приложения

  - страница для создание формул расчета цветов;
  - страница для настройки цветов страницы с таблицей студенты-виды работ;
  - страница для настройки цветов страницы с таблицей студенты-даты занятий;

5. Анализ результатов

  - страница с настройкой дат контрольных недель;
  - страница с результатами контрольных недель для студентов группы;
  - страница для настройки цветовой схемы предыдущей страницы.
